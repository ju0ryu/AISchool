HTML5 소개

html5가 인터넷 업계에서 알려지게 된 계기는 바로 구글의 웹 개발자 콘퍼런스인 스티브 잡스가 애플 아이폰에 플래시 탑재를 거부하면서 대응 기술로 홍보하기 시작
